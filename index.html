HTML

HyperText Markup Language

HTML é a linguagem padrão utilizada para criar e estruturar páginas na web e seu conteúdo.

HTML5

Linguagem de Marcação de Hipertexto

Ela permite a incorporação de textos, imagens, links e outros elementos multimídia, organizando e formatando o conteúdo para que seja exibido corretamente nos navegadores.

CSS

Cascading Style Sheets

CSS é uma linguagem usada para descrever a apresentação de um documento escrito em HTML ou XML, incluindo cores, layout e fontes.

CSS3

Folhas de Estilo em Cascata

O CSS da “vida” a aplicação web, melhorando a experiência do usuário. Ela permite separar o conteúdo da estrutura visual, facilitando a manutenção e a personalização do design das páginas web.

JavaScript

Linguagem de criação de scripts

JavaScript é uma linguagem de scripts, ou seja, de sequências de comandos, que permite a interatividade e o dinamismo nas páginas web.

JavaScript

O JavaScript pode ser utilizado para criar interfaces interativas do usuário nas páginas web, desenvolver a parte back-end de aplicativos, escrever sistemas operacionais, construir servidores de internet, usar bases de dados usando o JSSQL, desenvolver games aplicativos para celulares Android e iOS.
